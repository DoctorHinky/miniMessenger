# miniMessenger

## Benötigte Software

- [Node.js](https://nodejs.org/en/) (Version > 16)
- dotenv für umgebungsvariablen
- nodemon für live reload
- express für den server

  ## Installation

       1. Repository klonen
       2. In das Projektverzeichnis wechseln
       3. `npm install` ausführen

  ## Starten

  - `npm run start` / node index.js

  ## Besonderheiten

  - Es werden keine Daten gespeichert um vollkommene Anonymität zu gewährleisten
  - Die Lobby kann vom Host abgeschlossen werden wodurch keine weiteren Teilnehmer beitreten können
  - Der Host kann den Chat verlassen und ein neuer Host wird automatisch bestimmt

  - Es können Private Nachrichten versendet werden, die nur der bestimmte Client sehen kann

  - Es können Nachrichten an alle Clients versendet werden
